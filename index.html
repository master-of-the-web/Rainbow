Rainbow

Rainbow is a CSS Color Framework. 1343 Color Codes Based on Pantone Coated Colors. CSS classes are available for Background, Text, Border, SVG Fill & SVG Stroke Colors.

HOW TO USE

  1. Copy the css link and paste it inside your page <head></head> tag: <link href="rainbow/rainbow.min.css" rel="stylesheet">
  2. Then call css class in your design: <div class="bg-17 txt-1">Hello Rainbow :)</div>
  3. Open documentation/index.html file to view color list.
